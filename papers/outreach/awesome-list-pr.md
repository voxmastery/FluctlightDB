### FluctlightDB

Embedded memory **database engine** for AI agents — not a vector DB wrapper.

- **API:** `experience()` / `activate()` / `checkpoint()`
- **Claim:** third data model for agent memory (alongside relational + vector)
- **LoCoMo evidence recall:** 98.1% (full 10-conversation set, k=150)
- **License:** MIT
- **Links:**
  - LaTeX: https://github.com/voxmastery/FluctlightDB/tree/main/papers/arxiv-v1
  - HF: https://huggingface.co/Voxiesz/fluctlightdb-paper
